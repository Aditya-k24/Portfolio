# Oreliya – Full-Stack Jewelry E-Commerce Platform

A production-grade, scalable jewelry e-commerce application built as a TypeScript monorepo with full admin capabilities, Stripe payments, and comprehensive automated testing.

**GitHub:** [github.com/Aditya-k24/oreliya_webpage](https://github.com/Aditya-k24/oreliya_webpage)

---

## Architecture

Monorepo with three workspaces:
- **`apps/web-next`** — Next.js frontend with product browsing, cart, wishlist, checkout
- **`apps/api`** — Node.js + Express backend with full REST API
- **`packages/ui`** — Shared component library

---

## Core Features

### Shopping Experience
- Product catalog with variants (size, material, SKU), customizations (engraving, color), and sale pricing
- Cart, wishlist, address management, and order tracking
- Advanced search and filtering — category, tags, price range, sale status, stock availability, pagination
- Real-time inventory validation on checkout

### Authentication & Security
- JWT-based auth with access + refresh token rotation
- Role-based access control (RBAC) — `requireUser` / `requireAdmin` middleware
- bcrypt password hashing, Helmet security headers, CORS

### Admin Dashboard
- Analytics: daily sales (7-day), top products by quantity, low inventory alerts
- Deals management: percentage/fixed discounts, usage limits, date ranges
- Product customization presets and role management
- Full CRUD across all entities

### Payments
- Stripe integration with webhook handling and signature validation
